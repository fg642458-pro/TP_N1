## <center>Programación de APP para dispositivos móviles.</center>
### <center>Diagnostico</center>   
**1. ¿A qué nos referimos cuando hablamos de APP?**

 Grupos de programas que están orientado a un usuario final.

**2. ¿Qué es el Frontend y qué es el Backend? De una definición con sus palabras y explique cómo se comunican habitualmente.**

_Frontend es la parte que interactúa con el usuario, y el backend es la
información que se encuentra detrás de la interfaz, lo que no se ve. Se
comunican por medio de API._

**3. ¿Qué es una API REST? ¿Cuál es su función principal cuando desarrollamos una aplicación para dispositivos móviles?**

_Es un estándar que permite la comunicación entre en el front y el back._

**4. ¿Qué es el formato JSON? Escribe un ejemplo simple de cómo se vería un objeto "Alumno" con nombre, apellido y una lista de 3 notas en este formato.**

almacenamiento_

**5. ¿Qué es una librería (o biblioteca) y qué es un Framework? ¿Son lo mismo? Justifique su respuesta.**

_Una librería o biblioteca es un conjunto de funciones o clases enfocados en
una tarea que se puede usar las veces que sea. Un framework es la
estructura predefinida que proporciona la base para construir o como
organizar el código._


**6. ¿Qué es un sistema de control de versiones y para qué sirve? Explique qué sucede si dos desarrolladores modifican la misma línea de código en el mismo archivo y cómo se resuelve.**

_Un sistema de control de versiones es una herramienta de software que
registra los cambios realizados en archivos o códigos a lo largo del tiempo.
Sirve para gestionar el historial de modificaciones, permitir el trabajo
colaborativo simultáneo sin sobrescribir el trabajo ajeno, revertir errores y
respaldar proyectos de manera segura._
_En el caso de que dos desarrolladores modifiquen la misma línea de código,
prevalece la versión del administrador del proyecto._

**7. ¿Qué es la Programación Orientada a Objetos (POO)? Defina brevemente los conceptos de Clase, Objeto y Herencia.**

_En el caso de que dos desarrolladores modifiquen la misma línea de código,
prevalece la versión del administrador del proyecto._

**8. ¿Cuál es el propósito de los modificadores de acceso public y private en el contexto de la programación orientada a objetos?**

_El modificador private permite que los elementos solo sean visibles en la
propia clase. En cambio, public los elementos son visibles y accesibles
desde cualquier otra clase del proyecto._


**9. Explique las principales diferencias entre los sistemas operativos iOS (iPhone) y Android desde el punto de vista del desarrollo (lenguajes nativos, tiendas de aplicaciones y tipos de dispositivos). Además, mencione si conoce algún otro sistema operativo para móviles que exista o haya existido en el mercado.**

_Android: su hardware funciona en diversas marcas, utiliza el lenguaje de
Java, su tienda de aplicación es Play Store.
iOS: únicamente funciona con dispositivos Apple, el lenguaje que utiliza es
Swift, y su tienda de aplicación es App Store._
